# 赛博朋克城市飞行器 

一个使用 p5.js 开发的 3D 赛博朋克风格飞行探索游戏。

## 🎮 游戏特色

- **自由飞行**: 在 3D 赛博朋克城市中自由穿梭
- **霓虹美学**: 青色/粉色/紫色霓虹配色，营造未来都市氛围
- **收集玩法**: 收集漂浮的能量球获得分数
- **流畅操控**: 键盘 + 鼠标控制，支持加速飞行
- **鼠标视角**: 鼠标移动控制视角旋转，沉浸式体验

## 🚀 快速开始

### 方式一：直接打开

直接在浏览器中打开 `index.html` 文件即可游玩。

### 方式二：本地服务器

```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js http-server
npx http-server

# 然后访问 http://localhost:8000
```

### 方式三：在线部署（推荐）

**部署到 Vercel:**
```bash
# 安装 Vercel CLI
npm i -g vercel

# 部署
cd cyble-city
vercel
```

**部署到 Netlify:**
1. 访问 https://app.netlify.com/drop
2. 拖入 `cyble-city` 文件夹
3. 获得永久链接

## 🎨 设计理念

本项目采用模块化函数结构设计：

- **setup()**: 初始化场景、输入监听、生成城市
- **draw()**: 游戏主循环，渲染场景
- **update()**: 更新玩家位置、检测碰撞
- **generateCity()**: 程序化生成赛博朋克建筑
- **drawBuilding()**: 渲染单个建筑（带霓虹边缘和窗户）
- **drawCollectible()**: 渲染能量球（带旋转和脉动动画）
- **drawPlayer()**: 渲染飞行器（带尾焰粒子效果）

## 🌟 视觉效果

- **动态雾效**: 增强深度感，遮挡远处建筑
- **霓虹边缘**: 建筑轮廓发光，青/粉/紫三色随机
- **发光窗户**: 随机分布的黄色/青色窗户
- **能量球动画**: 旋转 + 脉动 + 轨道粒子
- **飞行器尾焰**: 动态粒子效果，加速时变粉色
- **地面网格**: 青色发光网格，营造科技感
