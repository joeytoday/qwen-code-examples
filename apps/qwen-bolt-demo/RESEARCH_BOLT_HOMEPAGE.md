## 四、核心功能区 - 输入框

**位置**：输入框顶部

## 八、交互设计

### 8.1 输入交互
1. 用户在输入框中输入需求描述
2. 可以选择 AI 模型
3. 可以添加附件
4. 点击 "Build now" 开始构建

### 8.2 导入交互
1. 点击 Figma 或 GitHub 按钮
2. 授权并选择要导入的内容
3. 自动生成项目

### 8.3 升级引导
- Token 使用提示
- Pro 版本升级链接
- 突出显示剩余额度

## 九、用户体验设计

### 9.1 信息层级
1. **最重要**：主标题 + 输入框（视觉中心）
2. **次要**：导航栏 + 导入选项
3. **辅助**：Token 提示 + 版本标识

### 9.2 引导流程
```
进入首页 → 看到主标题 → 注意到输入框 → 输入需求 → 选择模型 → 点击构建
```

### 9.3 空状态设计
- 占位符文本提供示例
- 清晰的行动号召（Build now）
- 多种输入方式（手动输入 / 导入）

## 十一、关键功能总结

### 11.1 核心功能
1. **AI 对话输入**：通过自然语言描述需求
2. **模型选择**：支持多种 AI 模型（如 Sonnet 4.5）
3. **快速构建**：一键开始项目构建
4. **导入功能**：支持从 Figma 和 GitHub 导入

### 11.2 辅助功能
1. **Token 管理**：显示剩余额度，引导升级
2. **对话历史**：保存和管理历史对话
3. **搜索功能**：快速查找历史对话
4. **账户管理**：设置、订阅、账户切换

### 11.3 社交功能
1. **社区链接**：Discord、LinkedIn、X、Reddit
2. **帮助中心**：提供使用帮助
3. **企业版**：面向企业用户的解决方案
