# City Running - 城市跑步游戏

第三人称城市跑步游戏，使用 Three.js 构建。

## 玩法

人物在城市街道上自动奔跑，玩家控制左右移动和跳跃，收集能量物品。

## 控制

- **A / ←**: 向左移动
- **D / →**: 向右移动
- **SPACE / ↑**: 跳跃
- **R**: 重置位置
- **C**: 切换相机视角

## 技术栈

- Three.js r163
- JavaScript (单文件)
- GLTFLoader (加载 Soldier 模型)

## 运行

直接在浏览器中打开 `index.html` 即可（需要本地服务器或禁用 CORS）。

推荐使用 VS Code 的 Live Server 插件：
```bash
# 安装 Live Server
code --install-extension ritwickdey.LiveServer

# 右键 index.html -> Open with Live Server
```
