# 02 - Code Review

## Goal

Analyze code differences between the current branch and the base branch, conducting a comprehensive code review.

## Checklist

### Difference Analysis ⭐⭐⭐

- [ ] View file change statistics
- [ ] View complete code differences
- [ ] Identify added/modified/deleted files

### Code Quality ⭐⭐⭐

- [ ] Check for obvious bugs
- [ ] Verify completeness of error handling
- [ ] Check for security vulnerabilities
- [ ] Verify code style consistency

### Logic Review ⭐⭐

- [ ] Understand the business purpose of changes
- [ ] Verify correctness of implementation logic
- [ ] Check boundary condition handling
